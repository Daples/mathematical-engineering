Programming Lab 1: Number Strings of an Arbitrary Size

TA: Juan Sebastián Cárdenas-Rodríguez

Introduction

A Deterministic Finite Automata (DFA) are finite automata that can only be at one state at a time. Furthermore, regular languages are exactly the ones described by finite automata [Hopcroft, Motwani and Ullman 2007].

The goal of this programming lab is to construct a function that finds how many strings of an arbitrary size belong to the language of a DFA.

Assignment (70%)

  • Download the GitHub repository 2020-2-lab1

  • (30%) Using the data type in the above repository define the function:

    stringsOfDFA :: DFA s c -> Int -> Int

    that takes a DFA and an integer number n and returns how many words of size n belong to the language of the given DFA.

    Suggestion: Use a similar strategy as Theorem 3.4 of the guide book of the course [Hopcroft, Motwani and Ullman 2007].

  • (30%) Oral explanation of your solution.

  • (10%)
    • Your lab will be tested using the -Wall, -Wmissing-signatures and -Wmissing-local-signatures.
    • Document (in English) your source code. The documentation should explain your solution for the assignment.
    • Improve your implementation with HLint ≥ 3.1.

  • The programming lab may either be solved on your own, or jointly with one other student taking the course.

  • You are to use GHC ≥ 8.10.1. See installing a proper Haskell distribution.

Examples

Using the DFA.hs module in the above repository, this is what a sample input-output looks like:

      GHCi> stringsOfDFA dfa0 0
      0

      GHCi> stringsOfDFA dfa1 0
      1

      GHCi> stringsOfDFA dfa1 10
      0

      GHCi> stringsOfDFA dfa2 0
      1

      GHCi> stringsOfDFA dfa2 3
      27

      GHCi> stringsOfDFA dfa2 9
      19683

      GHCi> stringsOfDFA dfa3 3
      1

      GHCi> stringsOfDFA dfa3 4
      0

      GHCi> stringsOfDFA dfa3 12
      1
